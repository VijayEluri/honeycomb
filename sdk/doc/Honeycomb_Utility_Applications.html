@HoneycombFullProductName@ Utility Applications

The @HoneycombProductName@ Client SDK includes Java utility classes to interface with the Honeycomb server. To display help for one of these utilities, run it from the command line without any arguments: java class_name arguments.

Archive

List, create, extract and delete named file archives from the @HoneycombProductName@.

Delete

Delete a metadata record from the @HoneycombProductName@.

Perf

When run in 'store' mode, perf will print Object IDs stored to stdout and print store stats to stderr. When run in 'retrieve' mode, perf will read Object IDs to retrieve from stdin, print Object IDs retrieved to stdout, and print retrieve stats to stderr. When run in 'stats' mode, perf will collect stats from other perf client and print aggregate stats to stdout.

Query

Query the @HoneycombProductName@ for metadata records matching a query string, returning OIDs.

QueryPlus

Query the @HoneycombProductName@ for metadata records matching a query string, returning Metadata values.

Retrieve

Retrieve data from the @HoneycombProductName@ for an OID.

Retrieve_std_out

Print data from the @HoneycombProductName@ to standard output for an OID.

Schema

Print the @HoneycombProductName@ schema.

Store

Storemetadata
