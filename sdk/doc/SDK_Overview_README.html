@HoneycombFullProductName@ Client SDK Overview

This is the client Software Development Kit (SDK) for the @HoneycombProductName@. If you have any questions regarding this SDK please refer to the contact information at the bottom of this document.

This SDK allows you to work with data and metadata stored on a @HoneycombProductName@. This document assumes you have a basic understanding of how the @HoneycombProductName@ uses data and metadata. For more information on the @HoneycombProductName@ data and metadata please see the section "Configuring Metadata and File System Views" in the @HoneycombFullProductName@ Administration Guide (below).

The @HoneycombProductName@ client SDK comes with an emulator (OpenEdition) that enables you to test client applications without having to connect to a @HoneycombProductName@. For further documentation on the emulator please refer to Emulator Setup and Configuration. Also refer to the @HoneycombProductName@ documentation to better understand the type of server being emulated.

The SDK provides client APIs for Java and C to store and retrieve data and to store, retrieve, query, and delete metadata records. For more information about the APIs refer to @HoneycombFullProductName@ Client API Reference Guide (below). Also for Java, see the javadoc API reference. There are also Java API Examples and C API Examples.
